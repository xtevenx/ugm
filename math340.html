Definition 1 (Binary Operation).

An internal binary operation defined on a set \(S\) is a mapping \(f:S\times S\to S\). In this document, we will refer to this simply as a binary operation.

The following are examples of binary operations:

  1. Addition on the set \(\mathbb{Z}\).

  2. Subtraction on the set \(\mathbb{Q}\).

  3. Multiplication on the set \(\mathbb{C}^{n\times n}\) (the set of \(n\times n\) matrices in the complex numbers).

The following are not binary operations:

  1. Subtraction on the set \(\mathbb{Z}_{\geq 0}\) (because \(0-1=-1\) is not in \(\mathbb{Z}_{\geq 0}\)).

  2. Division on the set \(\mathbb{Q}\) (because \(1/0\) is not defined).

Definition 2 (Ring).

A ring is a set \(R\) with two binary operations—addition \((+)\) and multiplication \((\cdot)\)—satisfying the following ring axioms:

  1. Addition is associative, meaning \((a+b)+c=a+(b+c)\) for all \(a,b,c\in R\).

  2. Addition is commutative, meaning \(a+b=b+a\) for all \(a,b\in R\).

  3. There is an identity element with respect to addition, meaning there exists \(0\in R\) satisfying \(a+0=a\) and \(0+a=a\) for all \(a\in R\).

  4. Every element has an inverse with respect to addition, meaning that for all \(a\in R\), there exists \(b\in R\) satisfying \(a+b=0\).

  5. Multiplication is associative, meaning \((a\cdot b)\cdot c=a\cdot (b\cdot c)\) for all \(a,b,c\in R\).

  6. There is an identity element with respect to multiplication, meaning there exists \(1\in R\) satisfying \(a\cdot 1=a\) and \(1\cdot a=a\) for all \(a\in R\).

  7. Multiplication is left-distributive over addition, meaning \(a\cdot (b+c)=a\cdot b+a\cdot c\) for all \(a,b,c\in R\).

  8. Multiplication is right-distributive over addition, meaning \((b+c)\cdot a=b\cdot a+c\cdot a\) for all \(a,b,c\in R\).
